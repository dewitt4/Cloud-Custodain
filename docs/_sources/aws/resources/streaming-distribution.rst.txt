.. _aws.streaming-distribution.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

.. _aws.streaming-distribution.actions.set-attributes:

set-attributes
++++++++++++++
Action to update the attributes of a distribution

:example:

.. code-block:: yaml

    policies:
    - name: enforce-streaming-distribution-logging
      resource: streaming-distribution
      filters:
        - type: value
          key: "Logging.Enabled"
          value: false
      actions:
        - type: set-attributes
          attributes:
            Logging:
                Enabled: true
                Bucket: 'test-enable-logging-c7n.s3.amazonaws.com'
                Prefix: ''
