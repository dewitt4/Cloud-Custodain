.. _aws.ops-item.actions.update:

update
++++++
Update an ops item.

: example :

Close out open ops items older than 30 days for a given issue.

.. code-block:: yaml

  policies:
   - name: issue-items
     resource: aws.ops-item
     filters:
      - Status: Open
      - Title: checking-lambdas
      - type: value
        key: CreatedTime
        value_type: age
        op: greater-than
        value: 30
     actions:
      - type: update
        status: Resolved
