.. _aws.firehose.actions.encrypt-s3-destination:

encrypt-s3-destination
++++++++++++++++++++++
Action to set encryption key a Firehose S3 destination

:example:

.. code-block:: yaml

        policies:
          - name: encrypt-s3-destination
            resource: firehose
            filters:
              - KmsMasterKeyId: absent
            actions:
              - type: encrypt-s3-destination
                key_arn: <arn of KMS key/alias>

.. container:: toggle

  

  .. raw:: html
     
    <div class="header docutils container" style=""></div>

  .. code-block:: yaml

    properties:
      key_arn:
        type: string
      type:
        enum:
        - encrypt-s3-destination
    required:
    - key_arn
