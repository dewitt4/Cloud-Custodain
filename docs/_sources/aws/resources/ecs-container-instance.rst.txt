taggable
++++++++
Filter ECS resources on arn-format
https://docs.aws.amazon.com/AmazonECS/latest/userguide/ecs-resource-ids.html
:example:

    .. code-block:: yaml

        policies:
            - name: taggable
              resource: ecs-service
              filters:
                - type: taggable
                  state: True

.. _aws.ecs-container-instance.actions.set-state:

set-state
+++++++++
Updates a container instance to either ACTIVE or DRAINING

:example:

.. code-block:: yaml

    policies:
        - name: drain-container-instances
          resource: ecs-container-instance
          actions:
            - type: set-state
              state: DRAINING

.. container:: toggle

  

  .. raw:: html

      state:
        enum:
        - DRAINING
