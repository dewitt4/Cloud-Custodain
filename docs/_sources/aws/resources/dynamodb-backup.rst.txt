.. _aws.dynamodb-backup.actions.delete:

delete
++++++
Deletes backups of a DynamoDB table

:example:

.. code-block:: yaml

        policies:
          - name: dynamodb-delete-backup
            resource: dynamodb-backup
            filters:
              - type: value
                key: BackupCreationDateTime
                op: greater-than
                value_type: age
                value: 28
            actions:
              - type: delete
