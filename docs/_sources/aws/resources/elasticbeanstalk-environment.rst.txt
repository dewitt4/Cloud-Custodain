.. _aws.elasticbeanstalk-environment.actions.terminate:

terminate
+++++++++
Terminate an ElasticBeanstalk Environment.

:Example:

.. code-block:: yaml

    policies:
      - name: eb-env-termination
        resource: elasticbeanstalk-environment
        filters:
          - type: marked-for-op
            op: terminate
        actions:
          - terminate
