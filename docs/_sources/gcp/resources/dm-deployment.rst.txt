delete
++++++
Deletes a deployment

:Example:

.. code-block:: yaml

    policies:
      - name: delete-deployments
        description: Delete all deployments
        resource: gcp.dm-deployment
        filters:
          - type: value
            key: name
            op: eq
            value: test-deployment
        actions:
          - delete
