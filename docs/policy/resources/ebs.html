delete

Schema:
{
  "force": {
    "type": "boolean"
  }
}
copy-instance-tags

Copy instance tags to its attached volume.

Schema:
{
  "tags": {
    "items": {
      "type": "string"
    },
    "type": "array"
  }
}

    "type": "number"

    "required": true,
    "type": "string"
  },
  "verbose": {
    "type": "boolean"
  }
}
